Sudoku Solver
-------------

Given a file such as **"example\_puzzle.txt"** with a sudoku board in the following format,

```
200804006
006000500
074000920
300040007
000305000
400060009
019000740
008000200
500608001
```

the program **"sudoku.cpp"** will attempt to solve the given sudoku puzzle. For now it can only solve the easiest of puzzles, but it will soon incorporate more advanced techniques to solve more challenging puzzles as well.
